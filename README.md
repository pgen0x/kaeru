# Kaeru

**Kaeru** is your ultimate tool for automating and managing multiple Roblox accounts on Android. It keeps your farming sessions running smoothly 24/7 by automatically handling crashes, disconnects, and window management.

## ✨ What It Does For You

### 1. **Keeps Your Games Running 24/7**
*   **Auto-Revival:** If Roblox crashes or closes, Kaeru detects it instantly and brings it back to life.
*   **Private Server Auto-Join:** Automatically joins your Private Server link when launching, so you're always in the right game.
*   **Anti-Stuck:** If an account gets stuck in the lobby, Kaeru force-stops it and tries again until it joins successfully.

### 2. **Organizes Your Screen**
*   **Smart Grid:** Automatically arranges your Roblox windows into a neat grid (like 2x2), so you can see everything at once.
*   **Focus Rotation:** Keeps all windows active and "awake" by rotating focus between them, preventing timeouts.
*   **Dashboard View:** Reserves a small space for a status dashboard, so you can monitor your farm's health without blocking the game view.

### 3. **Boosts Performance**
*   **Lag Reduction:** Automatically optimizes your device's CPU and RAM settings for smoother gameplay.
*   **Memory Cleaning:** Frees up memory when it gets low to prevent crashes.
*   **Debloating:** Turns off unnecessary system animations to save resources for Roblox.

## ⚙️ Customization (Optional)

You can customize settings by editing the `config.json` file located at `/storage/emulated/0/Kaeru/config.json` or setting environment variables:

*   `PRIVATE_SERVER_URL`: Your Private Server link.
*   `DELAY`: How long to wait on each window (default: 8 seconds).
