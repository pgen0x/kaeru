# Kaeru
**Kaeru** is your ultimate tool for automating and managing multiple Roblox accounts on Android. It keeps your farming sessions running smoothly 24/7 by automatically handling crashes, disconnects, and window management. Now with enhanced Cookie Login support!

## :sparkles: What It Does For You
### 1. **Keeps Your Games Running 24/7**
*   **Auto-Revival:** If Roblox crashes or closes, Kaeru detects it instantly and brings it back to life.
*   **Auto-Join:** Automatically joins your Private  / Public Server link when launching, so you're always in the right game.
*   **Auto-Reconnect:** Detects internet disconnects (Error 277, 279, etc.) and automatically rejoins the server.
*   **Anti-Stuck:** If an account gets stuck in the lobby, Kaeru force-stops it and tries again until it joins successfully.

### 2. **Advanced Account Management**
*   **Cookie Login (New!):** Seamlessly login to multiple accounts using cookies. No need to manually type passwords for every instance.
    *   **Export Cookies:** Save your currently logged-in accounts to a backup file.
    *   **Import Cookies:** Load accounts instantly from your backup files without typing passwords.
    *   **Logout All:** One-click logout for all accounts to clear sessions instantly.
*   **Organizes Your Screen**
    *   **Auto-Split:** Automatically arranges your Roblox windows into a neat grid.
    *   **Focus Rotation:** Rotates focus between windows to ensure CPU/RAM is distributed evenly across all instances.
    *   **Dashboard View:** Monitor farm health without blocking the game view.
    *   **Android 10 - 13+ Support:** Fully compatible with modern Android versions.

### 3. **Boosts Performance**
*   **Lag Reduction:** Automatically optimizes your device's CPU and RAM settings for smoother gameplay.
*   **Memory Cleaning:** Frees up memory when it gets low to prevent crashes.
*   **Debloating:** Turns off unnecessary system animations to save resources for Roblox.

### 4. **Discord Webhook Alerts**
*   **Executor Key Expired:** Get instant Discord notifications when your executor key expires.
*   **Account Recovery:** Automatic alerts when accounts come back online after issues.
*   **Real-time Monitoring:** Stay informed about your farm status from anywhere.

### 5. **Universal Game Support**
*   **Works with ANY Game:** Fully compatible with all Roblox experiences.
*   **Optimized for AFK:** Perfect for farming games like **Fish it**, **Fisch**, **The Forge**, **Anime Vanguards**, **Pet Simulator 99**, and more.
